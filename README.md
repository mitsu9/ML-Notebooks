# ML-Notebooks
Jupyter Notebooks for studying machine learning

# Environment setting
以下は自分が初めてやった時のログ
```
$ python --version
Python 3.5.0
$ pip3 install pipenv
$ pipenv --python 3.5.0 # 初期化
$ pipenv install jupyter 
$ pipenv run jupyter notebook
```

初めての人は以下の通り?
```
$ python --version
Python 3.5.0
$ pip3 install pipenv
$ pipenv install # Pipfileを見てインストール
$ pipenv run jupyter notebook
```
